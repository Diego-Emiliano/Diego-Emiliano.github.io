Matemáticas Básicas

Esta pagina pondrá ejercicios que se resuelven con matemáticas bascas sin usar formulas para mejorar tu comprensión y resolución de problemas.

Las matemáticas básicas consiste de sumas, restas, multiplicaciones, divisiones, elevación, raíz cuadrada, paréntesis y con orden de resolución de las operaciones.

Considerare que puedes resolver sumas, restas, multiplicación y divisiones de mas de 3 dígitos en el caso que no, te recomiendo obtener esta habilidad ya que a pesar de que sea muy fácil obtener y usar una calculadora hoy en día será mucho mas rápido y practico resolver así este tipo de operaciones y mas si tu trabajo esta relacionado con las matemáticas haz ejercicios simples sin combinar operaciones como 4000 o mas hasta que se te haga muy fácil hacer este tipo de operaciones.

Si han pasado años desde que aprendiste y usaste elevación, raíz cuadrada, paréntesis y (orden de resolución de operaciones) resolver o hacer operaciones con distintas operaciones es bastante probable que se te hallan olvidado ya que ese tipo de operaciones no se utilizan diariamente así que si no sabes estas operaciones lo siguiente te lo aclarara.

Elevación

La cantidad de veces que tienes que multiplicar por si mismo el numero ejemplo 5^3 esto quiere decir esto 5*5*5 = 125

Raíz Cuadrada

La raíz cuadrada es un numero multiplicado por si mismo que da el radicando (el numero adentro de la raíz cuadrada) ejemplo √9. La forma mas fácil de hacer raíces cuadradas es a ojo escogiendo y comprobando el numero multiplicado por si mismo que mas se acerca al radicando después si no hay números enteros que resuelvan la raíz cuadrada saca la calculadora si no puedes puedes sumar al numero mas cercano el (radicando-numero^2)/2*numero y te dará una aproximación mas cercana no el resultado. Es importante aclarar que todo resultado de una raíz cuadrada es - y + ya que al multiplicar un numero - o + por otro igual da siempre positivo.

Orden de resolución de Operaciones

Las operaciones con mas de una operación no se resuelven de izquierda a derecha tienen un orden y es este: paréntesis, corchetes, llaves, elevación, raíz cuadrada, multiplicación, división, suma y resta.

Paréntesis, corchetes y llaves

Son para que si quieres que primero se resuelva una parte de una operación antes que todo se colocan alrededor de ello los paréntesis, corchetes y llaves.

Representación de las operaciones y números

No usar coma o punto para separar cada 3 números nunca ya que dependiendo el país es con punto o coma y las computadoras no entienden números escritos de esa forma, para separar números enteros y decimales usa siempre el punto porque a pesar que mas países usen la coma para este fin todas las computadoras por estándar usan el punto para separar los decimales de los enteros y normalmente otra forma que veras los símbolos matemáticos en computadoras es esta:
Multiplicación *
División /
Raíz cuadrada √
Elevación ^ o también **

Ejercicios

Las matemáticas es una herramienta que puede servirte para muchas cosas como calcular la velocidad de un objeto o el área de un cuadrado, el como las uses depende de ti por eso no te voy a enseñar ese tipo de cosas pero te daré ejercicios de ecuaciones simples y cuadráticas, pero no debes resolverlas con formulas o cualquier procedimiento debes resolverlas con las operaciones básicas y pensamiento critico no sumes o multipliques sn pensar esperando patrones, sino analiza y evalúa para llegar a múltiples conclusiones para resolver las operaciones esto para que mejores tu comprensión matemática y resolución de operaciones. Copia estas ecuaciones cuadráticas muchas veces pero modificando solo los números excepto el 0 y las elevaciones y busca muchas ecuaciones simples y resuelves todas.

Ecuaciones simples

x*9 = 81

x*18-x*9 = 81

x*18-x*9+10-3 = 88

2*x*9-x*9+10-3 = 88

2*x*9/2-x*9+10-3 = 7

Ecuaciones cuadráticas

x^2-36=0

x^2-69=-5

x^2-7*x=0

x^2-9*x=0

2*x^2-8*x=-4*x

2*x^2-25*x=-9*x

2*x^2-8=0

3*x^2-147=0

2*x^2-4=4

3*x^2-49=-1
